# Personal Profile Website

This is a standalone static personal profile website built with:

- `index.html`
- `styles.css`
- `embedded-assets.js`
- `script.js`

## Preview

Open `index.html` in your browser.

## Customize

Edit the `profile` object at the top of `script.js`.

The quickest fields to update are:

- `name`
- `role`
- `bio`
- `email`
- `links`
- `projects`
- `current`

## Notes

- The layout is responsive for desktop and mobile.
- Fonts are loaded from Google Fonts.
- Images and the CV are embedded through `embedded-assets.js`, so the site can work without the `assets` folder.
- The starter content is easy to replace with your real details.
